What is Adios?

Adios is an advertisement blocker, it hides ads and block malicious scripts when you're browsing the Web using Safari. Adios also improves your browsing experience by blocking advertisement downloads, saving battery life and data consumption.

How does it work?

Adios manages a list of ad filters, it needs to download and process them in order to work. You need to configure Adios in order to block ads.

It's not working, what to do?

Adios only works on iPhone 5s or more recent, iPad mini 2 or more recent, iPad Air or more recent and iPod touch 6.
If you still see ads, check that Adios is activated in Settings > Safari > Content blockers and restart the configuration.
If you still have a problem, please send a private message to the Twitter account @Adios_app.

Can I configure Adios by myself?

If you want to select more precisely which lists to apply, long press "Configure Adios" on the home screen.
